Suddenly we realized that they were both heading toward the exact same point.  It looked as if they were going to collide!  It really felt like we were watching an action movie! PD))</sub></blockquote><sub><sub><i>source: [http://www.nuforc.org/webreports/reports/096/S96565.html](http://www.nuforc.org/webreports/reports/096/S96565.html)</i></sub></sub></sub><br>
