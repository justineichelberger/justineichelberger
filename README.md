The interesting part of these jets is I counted the time it took these jets to pass the same area that this object moved in.  The jets took approximately 12 seconds. The movements were quite sharp in direction.  It did not turn, but it just moved in the other direction for a period of time.  When comparing the distances between these passenger jets probably 30K feet, this object seemed a lot further away. My wife and I have never seen anything like this before.  We are in our 40's and don't typically follow this type of thing.  However, this did catch my attention and I know what I saw.
